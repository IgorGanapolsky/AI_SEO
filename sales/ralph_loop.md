# Ralph Loop

Status: operating loop defined. Not a claim of background execution or revenue.

## Objective

Run repeated blocks until the site is published, indexed, and at least one tracked affiliate conversion or paid service checkout lands.

## Loop

1. Verify revenue truth from Stripe and affiliate dashboards where API access exists.
2. Refresh `data/tools.csv` and `data/comparisons.csv` with current program facts.
3. Rebuild static pages with `ruby scripts/build_site.rb`.
4. Add or improve three comparison pages around buyer-intent keywords.
5. Publish one useful social post through Zernio for each active channel.
6. Record published URLs, failures, and next actions in `reports/gtm/`.
7. Never mark a post, email, or lead as sent unless the sending system returns a success URL, id, or visible sent confirmation.
